Resources Folder

This folder should contain:

1. icon.ico - Application icon file
   - Size: 256x256 pixels recommended
   - Format: .ico (Windows icon format)
   - Used for: Application icon, installer icon, shortcuts

To create an icon:
- Use an online converter (e.g., https://convertio.co/png-ico/)
- Or use a tool like GIMP, Photoshop, or IcoFX
- Save as icon.ico in this folder

If no icon is provided, the installer will use the default Windows icon.
