INSTALLATION NOTES
================================================================================

1. INSTALLATION DIRECTORY
   - Default: C:\Program Files\Transport Management System
   - You can choose a custom directory during installation
   - Ensure you have write permissions for the selected directory

2. DATA DIRECTORY
   - The application stores its database and files in the data\ folder
   - Default: [Installation Directory]\data
   - This directory requires write permissions
   - Make sure to backup this folder regularly

3. FIRST RUN
   - On first run, the application will:
     * Create the database structure
     * Load sample data for testing
     * Create necessary directories (logs, backups)
   - This process may take a few seconds

4. PERMISSIONS
   - The installer requires administrator privileges
   - The application needs write access to:
     * data\ folder (for database)
     * logs\ folder (for log files)
     * backups\ folder (for database backups)

UNINSTALLATION
================================================================================

To uninstall the application:

1. Go to: Settings > Apps > Installed Apps
2. Find "Transport Management System"
3. Click Uninstall

During uninstallation, you will be asked if you want to keep your data:
- Select "Yes" to keep your database and files
- Select "No" to completely remove all data

Note: If you keep the data, you can reinstall later without losing information.

================================================================================
BACKUP RECOMMENDATIONS
================================================================================

Important: Always backup your data regularly!

The application stores data in:
- Database: data\transport.db
- Backups: backups\ folder (automatic daily backups)

Manual backup:
1. Close the application
2. Copy the entire data\ folder to a safe location
3. Store backups on external drive or cloud storage

Restore from backup:
1. Close the application
2. Replace data\transport.db with your backup file
3. Restart the application
